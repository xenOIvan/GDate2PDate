## **Version 05.02.06** - November 16, 2025

### ✨ New Features
- Fix `may` english word  problem to prevent confusion with `May` (5th month)

## **Version 04.02.25** - November 16, 2025

### ✨ New Features
- Automatically converts Gregorian dates to Jalali (Persian) format on any webpage
- Supports multiple date formats (ISO, US, European, German, and textual dates)
- Detects and converts dynamically loaded content (AJAX support)
- Bilingual interface with full Persian (RTL) and English support
- Enable/disable toggle via extension popup
- Beautiful gradient UI with calendar-themed icons

### 🔧 Improvements
- Converts all date formats to unified YYYY/MM/DD output
- Minimal performance impact with optimized date detection
- Settings persist across browser sessions
- Works on all websites by default

### 🔒 Privacy
- All processing happens locally on your device
- No data collection or tracking
- No external network requests

### 📝 Notes
- Requires Chrome 88+ (Manifest V3)
- Extension is enabled by default after installation
- Click the extension icon to toggle on/off

## **نسخه 05.02.06** - ۲۶ آبان ۱۴۰۳

### ✨ ویژگی‌های جدید
- تبدیل خودکار تاریخ‌های میلادی به شمسی در هر صفحه وب
- پشتیبانی از فرمت‌های مختلف تاریخ (ISO، آمریکایی، اروپایی، آلمانی و متنی)
- تشخیص و تبدیل محتوای بارگذاری شده به صورت دینامیک (پشتیبانی AJAX)
- رابط کاربری دوزبانه با پشتیبانی کامل فارسی (راست‌چین) و انگلیسی

- ذخیره تنظیمات کاربر به صورت دائم
- فعال به صورت پیش‌فرض روی همه وب‌سایت‌ها

### 🔒 حریم خصوصی
- تمام پردازش‌ها روی دستگاه شما انجام می‌شود
- بدون جمع‌آوری داده یا ردیابی
- بدون ارتباط با سرورهای خارجی

- نیازمند کروم نسخه 88 به بالا
- افزونه پس از نصب به صورت خودکار فعال است
